# Jspec
Framework for testing web application  
[![NPM](https://nodei.co/npm/jspec.png)](https://nodei.co/npm/jspec/)

## Install Jspec
```shell
npm install jspec
```

## Update Jspec
```shell
npm update jspec
```

## Running Tests
Before you run tests, you may need to first install mocha module.
```shell
sudo npm install -g mocha
```

To run the tests:
```shell
npm test
```
